# Pixel Art Asset Checklist

**Project:** Onyx  
**Art Direction:** Simple top-down pixel art with a custom visual identity.

This checklist focuses on the MVP: one compact settlement, nearby gathering areas, one riskier gathering area, one 2-player dungeon, basic skills, combat, UI icons, and social/trade flows.

## Style Decisions To Make

- [ ] Tile size, for example 16x16, 24x24, or 32x32.
- [ ] Character sprite size.
- [ ] Camera scale/zoom target.
- [ ] Palette direction after lore is defined.
- [ ] Outline style: none, soft outline, or strong outline.
- [ ] Animation budget per character/enemy.
- [ ] UI icon size, for example 16x16 or 32x32.
